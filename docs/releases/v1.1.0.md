## Highlights

- **견적문의 폼 청소/이사 분리 + 가격표 자동 연동** — 청소 의뢰의 서비스 종류 드롭다운이 가격표(`price_items`) 항목 기반으로 동적 노출. `/price` 의 각 항목 클릭 시 견적문의 페이지로 즉시 이동하며 서비스 종류가 자동 채워짐.
- **모바일 견적문의 UX 강화** — 아이폰 HEIC 사진이 자동으로 JPEG 로 변환되고, 첨부 개수(4장) / 크기(25MB) / 형식 위반은 업로드 시점에 즉시 차단. 견적문의 메일 디자인 정제, 우편번호 검색·위치 기반 주소 입력 지원.
- **정책 v20260607 갱신 + 이전 버전 아카이브** — 처리위탁(Gmail SMTP) / 국외이전 / 이사 서비스·이용자 게시물 조항 신규. 페이지 상단 드롭다운으로 이전 시행 버전(`2026-03-23`)을 그대로 조회 가능.
- **외부 분석 도구(GA4·Clarity) 전면 제거** — 자체 DB 분석으로 단일 경로 일원화, 사이트 로딩 부담 감소 + 개인정보 처리 단순화.

## Added

- 견적문의 폼: 이미지 사전 검증(개수·크기·MIME·확장자) + iOS HEIC 자동 JPEG 변환 (`contact`)
- 정책 페이지 v2 (`/policy/privacy`, `/policy/terms`): 처리위탁·국외이전 안내, 이사 서비스·이용자 게시물 조항 신규. 이전 시행 버전을 드롭다운으로 조회하는 아카이브 라우트 (`/policy/{privacy,terms}/2026-03-23`) (`policy`)
- 홈 가격표 배너 + 관리자에서 안내문 동적 관리 (`price`)
- 자체 분석 수집 인프라 (Supabase 기반) + 행동 퍼널·랜딩·전화/CTA 추적 (`analytics`)

## Changed

- 견적문의 폼을 청소·이사 두 컴포넌트로 분리하고 공통 sub-component / 훅으로 재구성. 청소 폼의 서비스 종류 드롭다운이 **가격표(`price_items`) 항목 기반**으로 변경 (`contact`)
- `/price` 페이지에서 각 항목을 클릭하면 `/contact?service=<항목명>` 으로 이동하며 청소 폼 서비스 종류가 자동 채워짐 (`price`)
- 견적문의 메일 디자인 정제 + 주소 입력에 우편번호 검색 / 위치 기반 입력 강화, 견적 폼 안내 보강 (`contact`)
- 모바일 홈에서 사업자등록번호 영역의 iOS Safari 자동 전화번호 인식 차단 (`format-detection`) (`home`)
- 관리자 페이지: 콘텐츠/오늘 KPI/추이 차트 재설계, 정보관리 카드+모달 UI, 리뷰 리스트 통일 + 페이징, 자체 DB 기반 대시보드 (오늘/7일/30일) (`admin`)
- v1.0.1 의 의도된 액센트 색상 복원 — 평균 평점 큰 별 금색, "작업 진행" 단계 원 녹색 (`home`)

## Fixed

- v1.0.1 액센트 색상 회귀 — `refactor` 도중 단색 통일되며 사라진 평점 금색 별과 진행 완료 녹색 원 복원 (`home`)

## Removed

- 외부 분석 도구 GA4 + Microsoft Clarity 전면 제거 — 자체 DB 분석 단일 경로로 일원화 (`analytics`)
- IP hash 기반 rate limit 제거 — 분석 시스템 단순화 (`analytics`)
- 기존 단일 거대 `ContactForm.client.tsx` (480줄) 제거, 분리 컴포넌트로 대체 (`contact`)
- 소스코드 전체에서 설명형 주석 제거 (도구 디렉티브만 유지) (`internal`)
