## Added

- 푸터 최하단 "Designed by 노마드랩스" 텍스트를 외부 링크로 추가 — 클릭 시 새 탭으로 노마드랩스 사이트 이동. 도메인은 `NEXT_PUBLIC_NOMADLABS_URL` 환경변수로 주입 (`footer`)

## Changed

- 견적 문의 CTA를 공유 컴포넌트(`ContactCta`)로 추출하여 가격표·서비스·고객센터(FAQ) 페이지에서 동일하게 재사용. FAQ 페이지 하단에도 견적 문의 CTA 신규 노출, 추적 `content_id` 통일 (`contact`)

## Fixed

None

## Removed

- 견적문의 주소 입력의 현재 위치(GPS) 조회 기능 제거 — 주소 검색 API(Daum 우편번호)만 사용. 관련 `reverse-geocode` 라우트·인프라·스키마 및 테스트 삭제 (`contact`)

## Internal

- `middleware.ts` → `proxy.ts` 전환 — Next.js 16 proxy 파일 컨벤션 적용, deprecation 경고 해소 (`next`)
- quality 워크플로우 제거 (`ci`)
