## Highlights

청소클라쓰 공식 웹사이트 첫 출시. 방문자에게는 서비스 소개·Before/After·작업 후기·고객 리뷰·가격표·견적 문의를 한 페이지 흐름으로 제공하고, 사장에게는 GA4 데이터 기반 대시보드와 콘텐츠 CRUD를 단일 Next.js 애플리케이션으로 통합 제공합니다.

---

## 주요 기능

### 공개 사이트

- **랜딩 페이지** — 히어로, 서비스 그리드, Before/After 비교, 작업 프로세스, 블로그 리뷰, 고객 리뷰 캐러셀, FAQ, 견적 문의를 단일 페이지에 수직 구성
- **서비스 소개** — 청소·이사 카테고리별 상세 페이지. 포커스 포인트 기반 이미지 크롭으로 작업 품질을 직관적으로 전달
- **작업 후기** — 카테고리 필터 + 페이지네이션 기반 블로그 리뷰 아카이브
- **고객 리뷰 시스템** — 별점 평균 + 슬라이드 캐러셀 + 공개 리뷰 작성 (anon key 기반, 토큰 의존 제거)
- **가격표** — 12개 항목 그리드 + 최소 가격 표시 (`200,000원~`) + 현장 견적 항목 분리 표기
- **견적 문의** — 청소/이사 분기 Zod discriminated union 검증 + Nodemailer SMTP 발송 + 이미지 첨부 지원
- **정책 · RSS** — 개인정보처리방침, 이용약관, 도움말 FAQ + `/feed.xml` 신디케이션

### 관리자 대시보드 (`/admin`)

- **분석 대시보드** — GA4 Data API 9종 쿼리 병렬 조회 + Recharts 시각화 + 섹션별 새로고침
- **서비스 관리** — CRUD + 이미지 업로드(포커스 포인트 지정) + 드래그 정렬 (`useDragReorder` 공통 훅)
- **블로그 리뷰 관리** — 카드 큐레이션 + 링크/태그/이미지 인라인 편집
- **고객 리뷰 관리** — 별점/공개 토글/삭제 + 페이지네이션 + 익명 닉네임
- **작업 후기 관리** — 등록일 기준 정렬 + 최신순/오래된순 토글 (URL `?sort` searchParams)
- **가격표 관리** — 12개 시드 + CRUD + 드래그 정렬 + 가격 미정(`현장 견적`) 체크박스
- **FAQ 관리** — 카테고리별 추가/수정/삭제 + 노출 순서 조정
- **사이트 설정** — 상호·전화·이메일·SNS 링크 실시간 편집

### 보안 · 인증

- **4-layer admin 인증** — Middleware → admin/layout.tsx getUser → Server Action getUser → Supabase RLS
- **Zod 입력 검증** — 모든 server action에 uuid + form schema 검증
- **JSON-LD XSS 방어** — `<` → `<` 치환 일관 적용 + `JsonLdScript` 컴포넌트로 추출
- **이미지 매직 바이트 검증** — 클라이언트 위조 MIME/확장자 방어
- **Supabase RLS** — 공개 SELECT만 anon 허용, 쓰기는 authenticated

### SEO · 성능

- **메타데이터** — 모든 페이지에 title/description/og/twitter/canonical
- **sitemap.ts / robots.ts** — 동적 lastModified + 정책 명시
- **JSON-LD** — LocalBusiness, WebSite, Service, FAQPage, BreadcrumbList, ItemList(가격표) 6종
- **Next/Image** — focal point 기반 크롭, blur placeholder, 반응형 sizes, 캐시 TTL 1년
- **CSS 애니메이션** — 히어로 LCP 요소는 JS 비의존 CSS 키프레임, 아래 섹션은 IntersectionObserver

## Compatibility

- **Node.js**: 20.x+
- **Next.js**: 16.x (App Router, React 19)
- **Browser**: 모던 브라우저 (Chrome/Safari/Firefox/Edge 최신 안정 버전)
- **DB**: Supabase (PostgreSQL + RLS). 마이그레이션 17건 자동 적용
- **Auth**: Supabase Auth (관리자 단일 계정)

## Known Issues

- 토큰 시스템 제거 후 `submitPublicReview`는 anon key 기반 공개 POST 엔드포인트로, rate limiting / honeypot / CAPTCHA 미적용 — spam abuse 가능성 (별도 작업으로 도입 예정)
- 관리자 대시보드 GA4 데이터 새로고침 시 GA4 API 일시적 장애로 stale 데이터가 표시될 수 있음 (`safeAction`이 에러를 silent로 처리)
