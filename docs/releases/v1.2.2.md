## Changed

- 메인 헤더 로고를 컬러 버전으로 교체하고 표시 크기를 확대 — 중복 축소본(`logo-small.png`)을 제거하고 단일 `logo.png`로 통합, `next/image`가 표시 크기에 맞춰 최적화 (`asset`)
- 관리자 좌측 메뉴 순서를 재정렬하고 "리뷰 관리"를 "작업후기 관리"로 변경, "트래픽 분석"은 메뉴에서 빼고 대시보드 상단 링크 행(트래픽 분석·GA4 콘솔·Clarity 콘솔)으로 이동 (`admin`)

## Fixed

- 히어로 섹션에서 청소상담/이사상담 전화번호 중 하나만 입력된 경우 좌측에 치우치던 것을 중앙 정렬로 수정 (`hero`)

## Removed

- 관리자 대시보드의 "오늘 (KST 기준)" 통계 카드 섹션 제거 — 동일 지표를 바로 하단 일자별 추이 그래프에서 확인 가능 (`dashboard`)

## Internal

- 카드 제거에 따라 미사용 `getTodayStats` 쿼리·`TodayStats` 컴포넌트·타입·관련 테스트를 함께 정리 (`dashboard`)
