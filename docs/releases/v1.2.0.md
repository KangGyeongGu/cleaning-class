## Highlights

- 관리자 **트래픽 분석 대시보드**를 추가했습니다. Cloudflare 엣지(요청·위협·대역폭·캐시)와 GoAccess 오리진(인기 페이지 등) 지표를 **핵심 KPI · 일별 추이 · 상태/보안**으로 한 화면에서 조회하고, 더 상세한 통계는 **GoAccess 원본 보고서**로 확인합니다.
- 트래픽 데이터는 Cloudflare GraphQL·GoAccess 수집 스크립트(cron)로 모아 **서버 디스크에 영구 보관**하며, 기간 필터(7/30/전체)·CSV 내보내기·오래된 데이터 삭제를 지원합니다.
- 배포 빌드 인자(`NEXT_PUBLIC_NOMADLABS_URL`) 누락 수정 및 npm audit 취약점 8건을 해결했습니다.

## Added

- 관리자 트래픽 분석 대시보드(`/admin/traffic`) — Cloudflare 엣지·GoAccess 오리진 지표를 핵심 KPI(방문자·요청·페이지뷰·위협차단·서버에러·대역폭·캐시적중률) + 일별 추이 + 인기 페이지 + 상태/보안으로 조회. 기간 필터(7/30/전체), CSV 내보내기, 오래된 데이터 삭제, GoAccess 원본 HTML 보고서 열기 제공 (`admin`)
