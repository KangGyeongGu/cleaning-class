## Highlights

- 사용자 가시 동작 변화가 거의 없는 **내부 품질·구조 개선 릴리즈**입니다. 공통 UI 프리미티브(Button·Card·Table 등)와 공유 훅을 도입해 화면 전반의 중복 코드·스타일을 통합하고, 관리자 페이지를 인증 라우트 그룹으로 재구조화해 유지보수성과 일관성을 높였습니다.
- 메일 발송 라이브러리 **nodemailer를 보안 패치가 적용된 9.x로 업그레이드**했습니다. 견적문의 메일 발송 경로의 보안·안정성이 강화됩니다.

---

## Internal

- 공통 UI 프리미티브 레이어(`Button`/`ButtonLink`/`Card`/`Table`/`Badge`/`IconButton`/`Spinner`/`EmptyState`/`PublishToggle`)를 신설하고 39개 화면에 채택해 반복 스타일·마크업을 단일 컴포넌트로 통합 (`ui`)
- 견적폼·관리자 폼·목록 로직을 공유 훅(`useContactForm`/`useAdminForm`/`useListRowActions`/`usePagedList`/`useSingleImagePreview`)과 서버 래퍼(`with-auth-action`)로 통합하고, 디자인 시스템 시맨틱 클래스 채택으로 인라인 스타일 중복 정리 (`shared`)
- 관리자 페이지를 인증 라우트 그룹 `(protected)`으로 이동해 인증 가드를 `layout` 한 곳에서 일괄 처리 (공개 URL 불변) (`admin`)
- 미사용 `react-slick`·`slick-carousel` 의존성과 중복 컴포넌트(설명 섹션 5종·청소/이사 견적폼 분리본)를 제거 (`refactor`)
- nodemailer 9.x 보안 업그레이드 및 `npm audit` 취약점(@babel/core·js-yaml) 정리 (`deps`)
- 공유 모듈 단위·통합 테스트를 추가해 per-file 커버리지 100%를 유지하고, e2e 이미지 거부 테스트의 브라우저 모달 교착(데드락)을 수정 (`test`)
- 리팩토링 중 일시 도입된 견적폼 청소 `serviceType` 검증 회귀(가격표 DB 항목 거부)를 출시 전 발견·복원하고 회귀 방지 테스트 추가 (`schema`)
- 테스트·AI 운용 시스템(`.claude`)의 정합성 점검 및 정비 (`system`)
